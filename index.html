Work

  • DYRT...

    A Reddit style web app to serve up articles. It features the ability to choose a dark or light theme and infinite scrolling. Built with ReactJS, Styled Components and async await.

  • DYRT... RESTful API

    The backend for DYRT. Built with PostgreSQL, Express and Axios. HTTP request methods served are GET, POST, PATCH and DELETE.

  • Starry Skies?

    A personal project designed to calculate cloud cover during hours of darkness using 3rd party API data. A verdict for a good stargazing night is given. Built with ReactJS.

  • Chicken Tinder

    Can never decide where to eat? Try Tinder for restaurants! An APP built during the project phase of the Northcoders course as part of a team. Built with React, Apollo client provider, Material UI and Firebase.

  • Natural Events

    A data visualisation execise built with ReactJS, LeafletJS and chartJS. Using NASA's EONET API natural events such as wildfires and storms are marked on a map. The map is coded to centre on the most recent event.
